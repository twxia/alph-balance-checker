# Check ALPH Balance

## Typescript

```
# install
$ npm run install

# build
$ npm run build

# run built cli
$ npx run start <address>

# dev
$ npm run dev

# test
$ npm run test
```

## Golang

```
# build
$ go build ./cmd/main.go

# run built cli
$ ./main <address> # e.g. ./main 1HjTM3kG2M4J5BXnnNnn3TeVNhazmfmE5DEV2HRdiTZvg

# dev
$ go run ./cmd/main.go <address>

# test
$ go test ./internal/balance
```
